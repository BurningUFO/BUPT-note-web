    C++面向对象程序设计实践 - 课程概述

    课程信息

    • 课程名称: 面向对象程序设计实践（C++）
    • 学期: 2025-2026-2
    • 文档数量: 9个PDF讲义

    课程大纲

    01. Introduction（引言/课程介绍）

    • 课程概述与学习目标
    • C++语言发展历史
    • 面向对象编程思想简介
    • 课程安排与考核方式

    02. C++ Foundations（C++基础）

    • C++基本语法
    • 数据类型与变量
    • 运算符与表达式
    • 控制结构（条件、循环）
    • 函数基础
    • 输入输出流（iostream）

    03. Classes and Objects（类与对象）

    • 类的定义与声明
    • 成员变量与成员函数
    • 构造函数与析构函数
    • 对象的创建与使用
    • 封装性与访问控制（public/private/protected）
    • this指针

    04. C++ Programming（C++编程进阶）

    • 引用与引用传递
    • 函数重载
    • 默认参数
    • 内联函数
    • 常量与常量表达式
    • 类型转换

    05. Inheritance（继承）

    • 继承的概念与意义
    • 基类与派生类
    • 继承方式（公有继承、保护继承、私有继承）
    • 多重继承
    • 虚基类
    • 继承中的构造函数与析构函数

    06. Pointers（指针）

    • 指针基础概念
    • 指针与数组
    • 指针与函数
    • 指针与字符串
    • 动态内存分配（new/delete）
    • 智能指针简介

    07. Namespace（命名空间）

    • 命名空间的概念
    • 命名空间的定义与使用
    • using声明与using指令
    • 匿名命名空间
    • 命名空间的嵌套
    • 标准命名空间std

    08. Container（容器）

    • STL标准模板库简介
    • 序列容器（vector、list、deque）
    • 关联容器（set、map、multiset、multimap）
    • 容器适配器（stack、queue、priority_queue）
    • 迭代器使用
    • 常用算法（sort、find等）

    09. Exception Handling（异常处理）

    • 异常的概念
    • try-catch语句
    • throw抛出异常
    • 标准异常类
    • 自定义异常
    • 异常安全与资源管理

    学习建议

    1. 理论与实践结合: 每学完一个章节，建议编写相应的代码练习
    2. 重视基础: 02-04章是C++面向对象编程的基础，务必扎实掌握
    3. 理解思想: 不仅要掌握语法，更要理解面向对象的设计思想（封装、继承、多态）
    4. 多阅读代码: 学习优秀的开源C++项目，提升代码质量意识
    5. 善用STL: 熟练掌握标准模板库，提高开发效率

    推荐开发环境

    • IDE: Visual Studio、CLion、Code::Blocks、Dev-C++
    • 编译器: GCC、Clang、MSVC
    • 调试工具: GDB、Visual Studio Debugger

    参考资料

    • 《C++ Primer》（Stanley B. Lippman 著）
    • 《Effective C++》（Scott Meyers 著）
    • C++ Reference: https://en.cppreference.com/
    • ISO C++ 标准文档
